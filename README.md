# 🚀 Lets Build A Brand Of Yourself

An AI-powered analytics system that evaluates personal brand growth using behavioral metrics.

## Features

- Personal brand scoring algorithm
- Future regret risk prediction
- Behavior analysis
- Recommendation engine
- Habit tracking analytics
- Interactive Streamlit dashboard

## Tech Stack

- Python
- Streamlit
- SQLite
- Pandas
- Data Analytics

## How It Works

User inputs daily habits:

- Productivity
- Learning
- Networking
- Content Creation
- Health

The system calculates:

- Brand Score
- Future Regret Risk
- Behavior Insights
- Improvement Recommendations
